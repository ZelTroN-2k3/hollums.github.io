Instructions for downloading:

  • Extract the folder from the zip file and drag to your desktop.
  • For Macintosh:
    1. Open up the terminal (Shift+Cmd+A - Utilities - Terminal)
    2. Assuming your user name is "Alex" for example:
      Type: cd /Users/Alex/Desktop/Snake
    3. Next, type: make compile (you may need to download cmd line tools to do this action.
    For Windows:
    1. Open up the command prompt:
      Press Start, then type cmd in the search bar.
    2. Assuming your user name is "Alex" for example:
      Type: cd C:\Users\Alex\Desktop\Snake
    3. Next, type: make compile
  • To play against a "dumb" computer, type: make run_random
  • To play against an "intelligent" computer, type: make run_intelligent
  • To play against another player on the same computer, type: make run_human
  • To watch a "dumb" computer player play against an intelligent computer player, type: make run_randomvintelligent
  • To watch a "dumb" computer player against another "dumb" computer player, type: make run_randomvrandom
  • To watch an intelligent computer play against another intelligent computer player, type: make run_intelligentvintelligent
Game Commands:
help Displays list of commands
moves Displays the current players possible moves
quit Quit the current game
skip Not an actual move in classic reversi, but useful for testing
restart Restart the game.
rules Display the rules of the game.
